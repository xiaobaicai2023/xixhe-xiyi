# songshui-xiche-
洗车、上门洗车、洗车小程序、洗车APP、洗车预约、洗车项目、送水、送水小程序、送水系统、送水管理系统送水app、上门送水

# 技术栈

后台服务 springboot+mybatisplus+mysql
用户端 uniapp（vue语法）
管理后台 vue+elementUi

# 交付源码

提供源码，使用uniapp多端适配微信小程序 ，h5，公众号，安卓，ios
